# Lens

Git log analyzer.

## Installation

Install the [DMD 2 compiler](https://dlang.org/download.html) (using the MinGW setup option on Windows).

Build the executable with the following command line :

```bash
dmd -m64 lens.d
```

## Command line

```
lens
```

Analyzes the git log of the current folder.
